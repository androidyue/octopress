本人作为一个Markdown重度依赖患者，一直深受Markdown的高效写作的恩惠，一直在探索如何在浏览器中写Gmail邮件也可以使用Markdown这种高效语言。前端时间终于找到了答案，自己用着很棒。当然，好东西要分享才可以。遂作此文。
